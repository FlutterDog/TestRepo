## 3. Удаление Ethernet и Modbus TCP из раздела 1

### Было

Модуль должен обеспечивать обмен данными по Modbus RTU в объеме, предусмотренном встроенным программным обеспечением и картой регистров. Возможность обмена по Modbus TCP / Ethernet допускается только при наличии соответствующей аппаратной и программной реализации и должна определяться эксплуатационной документацией конкретного исполнения.

### Заменить на

Модуль должен обеспечивать обмен данными по Modbus RTU в объеме, предусмотренном встроенным программным обеспечением, картой регистров и эксплуатационной документацией.
