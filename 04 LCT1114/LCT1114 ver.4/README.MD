# Матрица развития LCT1114: Rev.2 / Rev.3 / Rev.3.1 / Rev.4 / Rev.4.1

## 1. Логика развития ревизий

| Ревизия          | Назначение                                                                                                                                                                                                              |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Revision 2**   | Текущая версия прибора. Используется как сдаваемая базовая версия. Обеспечивает учет механического и электрического ресурса, ограниченную регистрацию осциллограмм и работу в составе системы через внешний модуль LCP. |
| **Revision 3**   | Новая компактная ARM-платформа в форм-факторе модуля Lorentz. Цель — устранить ограничения Rev.2 по памяти, автономности, разрешению измерений, шумам и осциллографированию соленоидов.                                 |
| **Revision 3.1** | Программное развитие Rev.3 без существенного изменения аппаратной части. Основной акцент — расширенный анализ осциллограмм, COMTRADE, сервисные функции, улучшенная диагностика.                                        |
| **Revision 4**   | Отдельная rack-mount платформа, ориентированная на 19" исполнение, Linux/SoM, расширенные коммуникации, PRP, оптику, IEC 61850, управление/мониторинг дополнительных систем.                                            |
| **Revision 4.1** | Развитие Rev.4 с функциями AI/ML, анализом видеопотока, IP-камерами, интеллектуальным определением положения аппаратов и расширенной предиктивной диагностикой.                                                         |
