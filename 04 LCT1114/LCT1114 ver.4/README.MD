# Описание LCT1114: Rev.2 / Rev.3 / Rev.3.1 / Rev.4 / Rev.4.1

## 1. Логика развития ревизий

| Ревизия          | Назначение                                                                                                                                                                                                                                                                   |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Revision 2**   | Текущая версия прибора. Используется как сдаваемая базовая версия. Обеспечивает учет механического и электрического ресурса, ограниченную регистрацию осциллограмм и работу в составе системы через внешний модуль LCP.                                                      |
| **Revision 3**   | Новая компактная ARM-платформа в форм-факторе модуля Lorentz. Цель — устранить ограничения Rev.2 по памяти, автономности, разрешению измерений, шумам, ЭМС, осциллографированию соленоидов и связи.                                                                          |
| **Revision 3.1** | Программное развитие Rev.3 без существенного изменения аппаратной части. Основной акцент — расширенный анализ осциллограмм, COMTRADE, сервисные функции и улучшенная диагностика.                                                                                            |
| **Revision 4**   | Отдельная rack-mount платформа в 19" исполнении. База — Linux/SoM или аналогичная высокопроизводительная платформа. Назначение — расширенные коммуникации, PRP, оптика, IEC 61850, управление и мониторинг дополнительных систем, включая обогрев и управление выключателем. |
| **Revision 4.1** | Развитие Rev.4 с функциями AI/ML, анализом видеопотока, IP-камерами, интеллектуальным определением положения аппаратов и расширенной предиктивной диагностикой.                                                                                                              |

## 6. Основное обоснование перехода с Rev.2 на Rev.3

| Ограничение Revision 2                | Практическое последствие                                                       | Решение в Revision 3                                                 |
| ------------------------------------- | ------------------------------------------------------------------------------ | -------------------------------------------------------------------- |
| Ограниченный объем оперативной памяти | Нельзя хранить достаточный объем осциллограмм и выполнять расширенный анализ   | ARM Cortex-M7 с существенно большим объемом SRAM  ~512 кБ            |
| FRAM 8 КБ                             | Хранение только нескольких осциллограмм                                        | Event Recorder с RAM-буферами и Flash-хранилищем                     |
| 12-битный АЦП                         | Недостаточное разрешение и качество осциллограмм                               | 16-битный внешний ADC для главных токов                              |
| Нет High/Low gain                     | Нельзя одновременно получить широкий диапазон и хорошее разрешение малых токов | Два параллельных диапазона Low/High для трех фаз                     |
| Соленоиды без осциллограмм            | Невозможен анализ формы тока электромагнитов                                   | Осциллографирование соленоидов                                       |
| Нет RTC                               | Модуль не является полноценным автономным регистратором                        | Встроенные часы реального времени                                    |
| Зависимость от LCP                    | LCT не может самостоятельно хранить и обслуживать события                      | Автономная запись, хранение и выгрузка событий                       |
| SD-карта не задействована полноценно  | Нестабильное/неполное хранение данных                                          | Встроенная промышленная Flash-память на плате                        |
| Ethernet не доведен                   | Сетевые функции ограничены                                                     | W5500 + достаточная память для Modbus TCP / сервисной выгрузки       |
| Ограниченная ЭМИ-защита / шумы        | Плохое качество малых сигналов и ограниченная помехоустойчивость               | Переработанный аналоговый фронтенд, улучшенная ЭМИ-защита и разводка |
