## 1. Описание создаваемого Объекта

Прибор датчик плотности IGAS DM-91.

Датчик плотности IGAS DM-91 должен представлять собой стационарный одноблочный автоматический прибор непрерывного действия.

Прибор должен обеспечивать непрерывное измерение параметров рабочей среды, расчет плотности рабочей среды с учетом заданного состава газа и преобразование результатов измерений и расчетов в цифровой выходной сигнал для передачи внешним устройствам.

В рамках создаваемого Объекта должны быть предусмотрены исполнения прибора, отличающиеся составом измеряемых и расчетных параметров, в том числе:

- DM-910;
- DM-910h;
- DM-910 arctic.

В зависимости от исполнения прибор должен обеспечивать измерение абсолютного давления, температуры, расчет плотности рабочей среды, а также дополнительные измеряемые и расчетные параметры, предусмотренные эксплуатационной документацией, паспортом изделия и описанием протокола обмена.

Конструктивно прибор должен быть выполнен в корпусе, предназначенном для размещения измерительного узла, электронного блока и элементов электрического подключения.

Прибор должен иметь цифровой интерфейс передачи данных RS-485 с протоколом обмена Modbus RTU.

Настройка, калибровка, диагностика и считывание данных прибора должны выполняться с использованием внешнего программного обеспечения DM toolbox в объеме, предусмотренном документацией, передаваемой в составе результата работ.

## 2. Практическое назначение Объекта

Датчик плотности IGAS DM-91 предназначен для непрерывного измерения параметров рабочей среды, расчета плотности рабочей среды с учетом заданного состава газа и передачи измерительной информации внешним устройствам в цифровой форме.

Прибор должен применяться в системах контроля, регулирования и управления технологическими процессами.

Рабочими измеряемыми средами прибора должны являться газы, парогазовые и газовые смеси, а также иные среды, совместимые с конструкцией прибора и условиями применения, указанными в эксплуатационной документации.

### 5. Выбор технологий реализации

Технологии программирования, алгоритмы работы прибора, состав встроенного программного обеспечения, состав внешнего программного обеспечения и перечень передаваемых материалов определяются и указываются в Приложении № 1 к настоящему ТЗ.

В составе создаваемого Объекта должны быть предусмотрены встроенное программное обеспечение прибора и внешнее программное обеспечение IGAS Toolbox DM-91, предназначенное для настройки, диагностики, калибровки и сервисного обслуживания прибора.

Расчет калибровочных коэффициентов, подготовка таблиц поправок и формирование отчета о калибровке должны выполняться внешним программным обеспечением IGAS Toolbox DM-91. Встроенное программное обеспечение прибора должно обеспечивать прием, хранение и применение калибровочных коэффициентов и таблиц поправок при расчете и передаче измеряемых и расчетных параметров.

Внешнее программное обеспечение должно обеспечивать работу с прибором через интерфейс RS-485 с использованием протокола Modbus RTU.

В составе встроенного программного обеспечения прибора и внешнего программного обеспечения IGAS Toolbox DM-91 должны быть предусмотрены калибровочные функции в объеме, необходимом для соответствующего исполнения прибора.

Для прибора должны быть предусмотрены следующие основные калибровочные функции:

- калибровка нуля по каналу давления;
- калибровка нуля по каналу температуры;
- калибровка канала давления по набору контрольных точек;
- калибровка канала температуры по набору контрольных точек;
- калибровка канала влажности для исполнений прибора, в которых предусмотрен канал измерения влажности;
- учет температурной компенсации при обработке показаний канала давления;
- учет нелинейности измерительных каналов в пределах заданных контрольных точек;
- ввод, расчет, запись и хранение калибровочных коэффициентов;
- настройка минимальных и максимальных значений диапазонов измерения;
- передача калибровочных данных и коэффициентов в прибор;
- считывание калибровочных данных и коэффициентов из прибора;
- формирование отчета о калибровке.

Калибровка каналов давления, температуры и влажности должна выполняться по набору контрольных точек. Количество контрольных точек, порядок их задания, порядок расчета коэффициентов, порядок записи коэффициентов в прибор и порядок проверки результата калибровки должны определяться эксплуатационной документацией, руководством пользователя программного обеспечения и описанием протокола обмена.

Для канала давления должна быть предусмотрена возможность калибровки с учетом зависимости показаний давления от температуры.

Для канала температуры должна быть предусмотрена возможность калибровки по контрольным точкам с расчетом корректирующих коэффициентов.

Для канала влажности, в исполнениях прибора с измерением влажности, должна быть предусмотрена возможность калибровки по контрольным точкам с расчетом корректирующих коэффициентов.

Внешнее программное обеспечение должно обеспечивать отображение параметров калибровки, диапазонов, контрольных точек и результатов расчета коэффициентов в форме, достаточной для выполнения настройки, проверки и сервисного обслуживания прибора.

Отчет о калибровке должен формироваться в электронном виде и содержать идентификационные данные прибора, диапазон измерений, серийный номер, дату формирования отчета, использованные контрольные точки и рассчитанные калибровочные коэффициенты.

Калибровочные функции должны применяться в пределах диапазонов и условий, указанных в паспорте изделия, эксплуатационной документации, руководстве пользователя программного обеспечения и описании протокола обмена.

Подробное описание алгоритмов калибровки, форматов хранения коэффициентов, состава команд обмена, форматов отчетов и состава передаваемых файлов указывается в Приложении № 1, эксплуатационной документации, руководстве пользователя программного обеспечения и описании протокола обмена.
