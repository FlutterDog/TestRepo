# DM-910-x Firmware

Набор для сборки прошивки под AVR (atmega328p) без установки Atmel Studio.

## Структура папок

DM91-Firm\
├─ build_firmware.bat   — скрипт сборки прошивки
├─ config.c             — файл с настройками (адрес, скорость и т.п.)
├─ libDM-910-x.a        — библиотека вашего приложения
├─ toolchain\bin\     — локальный AVR-toolchain (avr-gcc, avr-objcopy и прочее)


## Требования

- Windows (батник и инструменты рассчитаны на Windows)


## Настройка

1. Откройте `config.c` в любом текстовом редакторе.
2. Измените значения констант (адрес Modbus, скорость порта, уровни сигналов и прочие).
3. Сохраните файл.


## Сборка прошивки

1. Перейдите в папку `FinalHex`.
2. Дважды щёлкните `build_firmware.bat` (или запустите его в консоли).
3. Скрипт выполнит:
   1. Компиляцию `config.c` → `config.o`
   2. Линковку с `libDM-910-x.a` → `firmware.elf`
   3. Генерацию Intel HEX → `firmware.hex`
   4. Удаление промежуточных файлов (`config.o`, `firmware.elf`)


## Результат

- **firmware.hex** — готовый файл прошивки для записи в MCU.
