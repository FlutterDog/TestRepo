# Программа и методика базовых испытаний печатной платы на производстве

## IGAS IR XBEAM Rev3

**Назначение:** документ устанавливает порядок выполнения **базовых производственных проверок** печатной платы/панели после изготовления перед передачей в сборку.

**Границы документа:**

* Документ предназначен для выявления типовых производственных дефектов и подтверждения пригодности платы к сборке.
* Документ **не является** программой квалификационных/сертификационных испытаний изделия.
* Диагностика неисправностей на уровне компонентов (поиск первопричины) в документ **не входит**.
* Проверки функционала собранного прибора выполняются по сервисной документации и/или через Toolbox и в настоящем документе не регламентируются.

**Результат испытаний:** одно из решений — **допустить к сборке** / **забраковать (в диагностику)**.

---

## 1. Объект контроля

* Печатная плата/панель газоанализатора **IGAS IR XBEAM Rev3**.
* Проверка выполняется **по панели целиком** (платы соединены штатными шлейфами).

---

## 2. Подготовка и оснастка

### 2.1 Оснастка

* Программатор **USBasp (ISP)** для записи загрузчика Optiboot.
* Источник питания **24 V DC** с ограничением тока.
* Интерфейс **RS-485** (USB–RS485 адаптер) и ПО для загрузки прошивки по RS-485.
* Средство экспресс-контроля температуры (пирометр/тепловизор) либо эквивалентный безопасный контроль отсутствия перегрева.

## 3. Параметры связи по умолчанию (после прошивки)

(после прошивки)

Используются для проверки обмена после загрузки основной прошивки.

| Параметр          | Значение                      |
| ----------------- | ----------------------------- |
| Интерфейс         | RS-485                        |
| Протокол          | Modbus RTU                    |
| Скорость          | 9600                          |
| Формат            | 8N1                           |
| Чётность          | none                          |
| Адрес устройства  | 1                             |
| Функция чтения    | Read Holding Registers (FC03) |
| Минимальный набор | HR 0..1 (2 регистра)          |

Примечание: **HR0** (концентрация) ожидаемо равен **0** при отсутствии подключённого сенсора. **HR1** (статус/ошибки) может быть ненулевым при отсутствии сенсора.

## 4. Программа и методика базовых испытаний (входной контроль)

### 4.1 Порядок выполнения и регистрация результатов

* Проверки выполняются **последовательно**.
* При отрицательном результате любого шага: **снять питание** (если применимо) и направить плату/панель **в диагностику**.

## 5. Примечание по дальнейшим проверкам (Toolbox)

Функциональные проверки собранного прибора (включая настройки и контроль параметров по сервисной утилите/Toolbox) выполняются по отдельной сервисной документации. Настоящий документ регламентирует только базовый входной контроль печатной платы/панели после производства.
