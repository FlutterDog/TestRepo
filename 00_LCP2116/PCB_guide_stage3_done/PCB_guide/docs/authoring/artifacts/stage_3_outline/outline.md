ℹ️ Этот файл фиксирует каркас финального `docs/release/board_guide.md` и задаёт content contract для Stage four.

ℹ️ Каноническая идентичность для текущего прохода принята по решению автора: `LCP2116 Rev.7.1`.

⚠️ Scope этой структуры — только аппаратная часть платы и product-level claims, подтверждённые freeze-комплектом и Stage two.

⚠️ Для Stage four заранее запрещено:

* возвращать в release конфликт `LCP2216` / `rev 7.2` как равноправную идентичность;
* обещать неподтверждённую гальваническую изоляцию интерфейсов;
* заявлять численный диапазон входного питания сверх того, что безопасно подтверждено manual-level источником;
* смешивать hardware-описание с прошивкой, HMI, конфигурационными файлами и алгоритмами работы шкафа.
