# FIGURE BRIEF TEMPLATE

## Назначение документа

Этот файл нужен, если planned figures / tables требуют отдельного brief от автора.

Используйте его, когда в `outline.md` уже понятно, что asset нужен, но сам файл ещё не подготовлен.

---

## Как использовать

Один brief — один будущий asset.

Заполняется кратко. Цель — не нарисовать фигуру словами, а дать понятное ТЗ на её подготовку.
