# FILE MAP TEMPLATE

## Document role

ℹ️ Этот файл — канонический слой трассируемости между инженерными утверждениями и Source of Truth.

❗ Одна запись = одно проверяемое утверждение.

---

## Status legend

ℹ️ FACT — подтверждено источником.

ℹ️ ASSUMPTION — допущение, явно отмеченное и контролируемое.

ℹ️ UNKNOWN — пока не подтверждено; требуется источник или отдельная проверка.

---

## Entry format

Скопируй этот блок для каждой новой записи.

### Statement ID

* `<FM-001>`

### Status

* `<FACT | ASSUMPTION | UNKNOWN>`

### Statement

* `<one precise engineering statement>`

### Source anchor

* Anchor: `<page | sheet | refdes | net | token | section>`
