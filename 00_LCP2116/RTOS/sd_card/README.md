# Пример содержимого microSD

Скопируйте нужные TXT-файлы из этого каталога в корень microSD контроллера.

## X2X

`X2X.TXT` хранится отдельно в рабочем комплекте и задаёт состав системных модулей.

## FieldSensor S1–S4

```text
BAUD.TXT
PARITY.TXT
```

Порядок значений: S1, S2, S3, S4.

## Ethernet

```text
ETH1: IP.TXT, SUBNET.TXT, GATE.TXT
ETH2: IP2.TXT, SUBNET2.TXT, GATE2.TXT
```

Все числовые файлы используют формат:

```text
количество значений
значение 1
значение 2
...
```

Примеры в каталоге настроены на:

```text
S1-S4: 9600 8N1
ETH1: 192.168.1.1/24
ETH2: 192.168.1.2/24
Gateway: 192.168.1.254
Modbus TCP port: 502
```

Перед копированием сетевых файлов проверьте, что эти IP-адреса не конфликтуют с другими устройствами сети.
