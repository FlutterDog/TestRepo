# Управление X2X через USB-консоль

Подсистема X2X работает неблокирующе внутри базовой задачи LCP. Команды, меняющие конфигурацию или состояние опроса, не обрывают уже начатый Modbus RTU кадр.

## Просмотр состояния

```text
x2x
```

Команда выводит:

- результат загрузки `X2X.TXT`;
- состояние построения реестра;
- количество модулей и текущий slave;
- владельца UART0;
- результат software self-test CRC;
- аппаратные ошибки UART;
- состояние связи и счётчики каждого модуля;
- основные данные модулей;
- состояние последней осциллограммы LCT.

## Повторная загрузка X2X.TXT

```text
x2x reload
```

Если X2X в этот момент свободен, файл проверяется и применяется сразу:

```text
x2x reload: ok
```

Если выполняется цикл опроса модуля, команда возвращает:

```text
x2x reload: apply pending
```

Это штатное состояние. Новая конфигурация будет прочитана и применена после завершения текущего полного цикла модуля. Проверить завершение можно повторной командой:

```text
x2x
```

Ожидается:

```text
reload_pending=no
```

Невалидный новый файл не заменяет ранее работающую конфигурацию. Исключение — внутренняя ошибка построения реестра, которая указывает на дефект прошивки и выдаётся как `registry build failed`.

## Приостановка опроса

```text
x2x pause
```

При свободной шине ответ:

```text
x2x polling: paused
```

Во время активного цикла:

```text
x2x polling: pause pending
```

Сервис завершит текущий цикл модуля, включая необходимые операции LCT, и только затем остановится. Это исключает обрыв UART-передачи и зависание DE в режиме передачи.

Состояние:

```text
paused=yes
pause_pending=no
```

## Возобновление

```text
x2x resume
```

Опрос продолжается с текущей конфигурацией:

```text
x2x polling: running
```

## Сервисная установка LDO

```text
x2x ldo SLAVE VALUE
```

Пример:

```text
x2x ldo 1 255
```

Команда изменяет локальный выходной регистр экземпляра LDO1118. Значение передаётся модулю при следующем цикле его опроса. Команда предназначена для аппаратной проверки и не заменяет будущую прикладную логику управления выходами.
