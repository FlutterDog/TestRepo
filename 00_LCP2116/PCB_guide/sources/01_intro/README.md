1. Общая информация

Устройство LCP2116 – это процессорный модуль из линейки продуктов серии Lorentz, выполненный в пласти-
ковом корпусе для установки на DIN-рейку.
Основные функции устройства:
– Управление и контроль модулей входов/выходов (I/O) серии Lorentz через последовательную шину X2X.
– Цифровая связь с внешними устройствами с помощью RS485 и Ethernet
– Взаимодействие с сенсорными панелями вывода и ввода информации
– Ведение журналов событий с записью на uSD карту.
Модуль управляется операционной системой реального времени IGAS RT. Выполнен на базе процессора ARM
Сortex M3. Файлы конфигурации сохраняются в энергонезависимой памяти на флэш-карте (выдвижная карта uSD-card).
Для сохранения точного времени во время отключения питания используются часы реального времени с резервиро-
ванием от батареи CR2032.
Модуль связи поддерживает следующие интерфейсы:
– Порты связи 1...7 последовательные интерфейсы RS485.
– Ethernet интерфейсы 1 и 2 (E1 & E2): 10/100BaseT с разъемами типа RJ45.
– Интерфейс USB 2.0 для диагностики и технического обслуживания.
– Внутренняя шина X2X для локального обмена данными между локальными и удаленными модулями входов/выходов
