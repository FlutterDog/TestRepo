
# IGAS-IR

**Оптический газовый анализатор ИК
(Near InfraRed Gas Analyzer)

---

Справочник по внутреннему программному обеспечению ATmega328

---


## ℹ️ Примечания


Прошивка для микроконтроллера ATmega328P (семейство IGAS).  
Ориентирована на сборку в AVR toolchain.

## Состав репозитория

- `app/` — прикладная логика (алгоритмы, обработчики, карта регистров Modbus, глобальные переменные проекта).
- `hal/` — аппаратная совместимость (Serial/SPI/GPIO/tick), shim-слой для минимальной зависимости от Arduino.
- `Libs/` — локальные библиотеки и совместимые обёртки (EEPROM, TimerOne и т.п.).
- `docs/` — выходная документация Doxygen (генерируется, не редактировать вручную).

## Документация (Doxygen)

Документация генерируется из исходников и комментариев Doxygen.  
Файлы проекта сохранены в UTF-8 (это требование для корректного вывода русскоязычных описаний).

### Требования
- Doxygen установлен и доступен в PATH:
  - проверка: `doxygen --version`
