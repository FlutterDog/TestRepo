# REFERENCE SOURCES TEMPLATE

## Назначение документа

Этот файл перечисляет **разрешённые официальные внешние источники**, которые допустимо использовать на Stage two–Stage five для характеристик компонентов, модулей и типовых подсистем.

❗ Этот файл не заменяет freeze-комплект платы.
Он дополняет его.

---

## Entry format

### Reference ID

* `<RS-001>`

### Source type

* `<datasheet | reference manual | product brief | module manual | official app note>`

### Object

* `<component / module / subsystem name>`

### Canonical location

* `<sources/extras/references/<file> | official URL | both>`

### What may be taken from this source

* `<key characteristics | supported modes | memory size | interface speed | electrical limits | other>`

### What may NOT be claimed from this source automatically

* `<board-level capability without board confirmation | performance promises | unsupported functions | other>`

### Status

* `<allowed | allowed with caution | blocked>`
