## 1. Общие сведения о плате

Плата `LCP2116` предназначена для процессорного модуля `Lorentz LCP2116`.

`LCP2116` выполняет функции центрального контроллера системы `Lorentz`. Плата управляет модулями ввода/вывода по внутренней последовательной шине `X2X`, обеспечивает обмен с внешними устройствами по `RS485` и `Ethernet`, взаимодействует с панелями оператора, хранит конфигурационные данные и журналы на `microSD`, а также поддерживает системное время с помощью `RTC` с резервным питанием от батареи `CR2032`.

### 1.1. Технические характеристики

#### Основные характеристики

| Параметр | Значение |
|---|---|
| Интерфейсы | `7 x RS485`, `2 x Ethernet`, `1 x USB`, `1 x X2X` |
| Назначение модуля | Центральный процессор |
| Поддержка резервирования | Да |
| Поддержка экранов отображения | Да |
| Потребление питания | `1 Вт` |
| Степень защиты модуля | `IP20` |

#### Диагностика

| Параметр | Значение |
|---|---|
| Состояние процессора | Да, светодиодный индикатор |
| `RS485` передача данных | Да, светодиодный индикатор |
| `Ethernet` | Да, светодиодный индикатор |
| Питание | Да, светодиодный индикатор |
| `X2X` сеть | Да, светодиодный индикатор |

#### Параметры шины питания модулей

| Параметр | Значение |
|---|---|
| Входное напряжение | `24 В -20%`, постоянный ток |
| Ток | Максимум `1 А` |
| Предохранитель | Интегрированный самовосстанавливающийся |
| Защита от неверной полярности | Да |

#### Контроллер и память

| Параметр | Значение |
|---|---|
| Процессор | `ARM` |
| Частота процессора | `84 МГц` |
| `SRAM` | `100 кБ` |
| `RAM` буфер | `4 кБ ECC` |
| Интегрированные вводы-выходы | Да, обработка в фоновых процессах |
| Поле переменных | `512 кБ FRAM` |
| Минимальное время цикла | `2 мс` |

#### Память приложений

| Параметр | Значение |
|---|---|
| Тип носителя | `2 ГБ microSD` |
| Перезаписываемый объём | `40 ТБ` |
| Количество циклов перезаписи | `20 000` |

#### Интерфейсы связи

| Интерфейс | Параметр | Значение |
|---|---|---|
| Интерфейс `0` | Стандарт | `RS485` |
| Интерфейс `0` | Длина линии | `900 м` |
| Интерфейс `0` | Скорость обмена данными | Макс. `1,0 Мбит/с` |
| Интерфейс `0` | Подключение | `2`-проводный клеммник |
| Интерфейсы `1...6` | Стандарт | `RS485` |
| Интерфейсы `1...6` | Длина линии | `1200 м` |
| Интерфейсы `1...6` | Скорость обмена данными | Макс. `110 кбит/с` |
| Интерфейсы `1...6` | Подключение | `2`-проводный клеммник |
| Интерфейсы `7...8` | Стандарт | `Ethernet` |
| Интерфейсы `7...8` | Длина линии | До `100 м` |
| Интерфейсы `7...8` | Скорость обмена данными | `10/100 Мбит/с` |
| Интерфейсы `7...8` | Подключение | `1 x RJ45` |
| Интерфейсы `7...8` | Физический уровень | `10BASE-T/100BASE-TX` |
| Интерфейсы `7...8` | Полудуплекс | Да |
| Интерфейсы `7...8` | Полный дуплекс | Да |
| Интерфейс `9` | Стандарт | `USB 2.0` |
| Интерфейс `9` | Подключение | `USB Type B` |

#### Электрические параметры

| Параметр | Значение |
|---|---|
| Изоляция | Все интерфейсы полностью изолированы друг от друга |

#### Условия эксплуатации

| Параметр | Значение |
|---|---|
| Монтаж в горизонтальном положении | Да |
| Монтаж в вертикальном положении | Да |
| Степень защиты | `IP20` |
| Температура эксплуатации | от `-25` до `60 °C` |
| Температура хранения | от `-40` до `85 °C` |
| Температура транспортировки | от `-40` до `85 °C` |
| Относительная влажность при эксплуатации | от `5` до `95%`, без конденсации |
| Относительная влажность при хранении | от `5` до `95%`, без конденсации |
| Относительная влажность при транспортировке | от `5` до `95%`, без конденсации |
