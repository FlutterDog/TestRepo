## 1. Общие сведения о плате

Плата `LCP2116` представляет собой процессорный модуль `Lorentz LCP2116` и используется в качестве центрального контроллера системы `Lorentz`. Модуль обеспечивает управление модулями ввода-вывода по внутренней последовательной шине `X2X`, обмен с внешними устройствами по интерфейсам `RS485` и `Ethernet`, взаимодействие с панелями оператора, а также хранение конфигурационных данных и журналов на карте `microSD`.

В состав платы входит узел часов реального времени `RTC` с резервным питанием от батареи `CR2032`, что сохраняет системное время при отключении основного питания. По совокупности функций модуль выполняет роль центрального вычислительного и коммуникационного узла системы.
